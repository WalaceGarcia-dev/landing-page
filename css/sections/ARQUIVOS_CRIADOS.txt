================================================================================
                        COMO USAR OS ARQUIVOS
================================================================================

OPÇÃO 1: Importar cada arquivo no HTML
-------------------------------------------
<link rel="stylesheet" href="css/sections/slider.css">
<link rel="stylesheet" href="css/sections/sobre-nos.css">
<link rel="stylesheet" href="css/sections/especialidades.css">
<link rel="stylesheet" href="css/sections/servico-individual.css">
<link rel="stylesheet" href="css/sections/testimonials.css">
<link rel="stylesheet" href="css/sections/galeria.css">
<link rel="stylesheet" href="css/sections/artigos.css">


OPÇÃO 2: Usar o main.css (RECOMENDADO)
-------------------------------------------
No HTML, adicionar apenas:
<link rel="stylesheet" href="css/main.css">

O arquivo main.css já contém os imports de todos os arquivos modulares.


OPÇÃO 3: Usar imports em CSS
-------------------------------------------
Criar um arquivo css/custom.css com:

@import url('sections/slider.css');
@import url('sections/sobre-nos.css');
@import url('sections/especialidades.css');
@import url('sections/servico-individual.css');
@import url('sections/testimonials.css');
@import url('sections/galeria.css');
@import url('sections/artigos.css');

E importar apenas este arquivo no HTML.
